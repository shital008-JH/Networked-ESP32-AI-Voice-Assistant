## ğŸš€ Project Overview

The **Networked ESP32 AI Voice Assistant** is designed as a smart embedded device capable of voice interaction.

The system workflow:

1. User speaks into the microphone
2. ESP32 captures audio input
3. Audio is transmitted over Wi-Fi to a backend server
4. The server performs AI/LLM-based processing
5. The assistant sends back a response
6. Output is played/displayed in real time
